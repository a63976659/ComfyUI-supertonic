# ComfyUI-supertonic
  文本转语音。
  在运行示例之前，请下载ONNX模型和预设声音，并将它们放在assets目录中：在 ComfyUI\custom_nodes\ComfyUI-supertonic 目录中 cmd 输入 hf download Supertone/supertonic --local-dir 你的comfyui地址\custom_nodes\supertonic\assets 或者 手动下载 https://huggingface.co/Supertone/supertonic/ 所有文件到ComfyUI\custom_nodes\ComfyUI-supertonic\assets目录中


  插件来源于开源项目https://github.com/supertone-inc/supertonic
